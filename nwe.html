توابع کلیدی SFML

ایجاد شده توسط محسن سعیدی

استاد: ایمان اشکاوند

مقدمه: SFML چیست؟

SFML مخفف Simple and Fast Multimedia Library است. این یک کتابخانه رایگان و متن‌باز برای برنامه‌نویسی چندرسانه‌ای (multimedia) در زبان C++ است. SFML به طور گسترده برای ساخت بازی‌های دوبعدی، اپلیکیشن‌های گرافیکی، و برنامه‌هایی که با صدا، تصویر، و ورودی‌های کاربر (مانند کیبورد و موس) سر و کار دارند، استفاده می‌شود.

ویژگی‌های کلیدی SFML عبارتند از:

  • سادگی: یادگیری و استفاده از SFML آسان است، به خصوص برای مبتدیان.
  • سرعت: SFML با استفاده از شتاب‌دهنده سخت‌افزاری (Hardware Acceleration) عملکرد بالایی ارائه می‌دهد.
  • چندسکویی (Cross-platform): برنامه‌های نوشته شده با SFML می‌توانند بر روی سیستم‌عامل‌های مختلفی مانند ویندوز، لینوکس، و macOS اجرا شوند.
  • شیءگرایی: SFML با یک API شیءگرا طراحی شده است که کدنویسی را سازمان‌یافته‌تر می‌کند.
  • ماژولار بودن: SFML از چندین ماژول تشکیل شده است (System, Window, Graphics, Audio, Network) که می‌توانید فقط ماژول‌های مورد نیاز خود را استفاده کنید.

SFML یک انتخاب عالی برای شروع توسعه بازی و گرافیک در C++ است و جامعه کاربری فعالی دارد که منابع آموزشی و پشتیبانی خوبی ارائه می‌دهند.

نصب و راه‌اندازی SFML

برای استفاده از SFML در پروژه‌های C++ خود، ابتدا باید آن را دانلود و در محیط توسعه خود (مانند Visual Studio) پیکربندی کنید. در ادامه مراحل کلی نصب و راه‌اندازی SFML (نسخه 2.5.1 برای مثال) با MinGW (برای کامپایلر GCC) و Visual Studio آمده است:

مرحله ۱: دانلود فایل‌های مورد نیاز

  • دانلود کتابخانه SFML:
    • به وب‌سایت رسمی SFML به آدرس sfml-dev.org/download.php بروید.
    • نسخه مناسب SFML را برای کامپایلر و معماری سیستم خود دانلود کنید (مثلاً برای Visual Studio 2017 64-bit یا GCC 7.3.0 MinGW (DW2) 32-bit).
    • فایل دانلود شده (معمولاً یک فایل فشرده .zip یا .tar.gz) را استخراج کنید. پیشنهاد می‌شود محتویات آن را در یک مسیر مشخص و بدون فاصله در نام، مانند C:\SFML-2.5.1 کپی کنید.
  • دانلود کامپایلر (در صورت نیاز):
    • اگر از Visual Studio استفاده می‌کنید، کامپایلر C++ به همراه آن نصب شده است.
    • اگر از کامپایلرهای مبتنی بر GCC مانند MinGW استفاده می‌کنید، مطمئن شوید که نسخه سازگار با SFML دانلود شده را نصب کرده‌اید (مثلاً MinGW-w64 با GCC 7.3.0).

مرحله ۲: ساخت پروژه در محیط توسعه

یک پروژه جدید C++ در محیط توسعه خود (IDE) ایجاد کنید. برای مثال، در Visual Studio یک "Empty Project" بسازید.

مرحله ۳: تنظیمات پروژه (مثال برای Visual Studio)

  1. یک فایل main.cpp به پروژه خود اضافه کنید.
  2. روی نام پروژه در Solution Explorer راست کلیک کرده و `Properties` را انتخاب کنید.
  3. تنظیمات کلی:
    • `Configuration`: `All Configurations` (یا `Debug` و `Release` را جداگانه تنظیم کنید).
    • `Platform`: `x64` یا `x86` (بسته به نسخه SFML دانلود شده).
  4. مسیرهای Include:
    • به `Configuration Properties > C/C++ > General > Additional Include Directories` بروید.
    • مسیر پوشه `include` کتابخانه SFML را اضافه کنید (مثلاً `C:\SFML-2.5.1\include`).
  5. مسیرهای Library:
    • به `Configuration Properties > Linker > General > Additional Library Directories` بروید.
    • مسیر پوشه `lib` کتابخانه SFML را اضافه کنید (مثلاً `C:\SFML-2.5.1\lib`).
  6. وابستگی‌های Linker:
    • به `Configuration Properties > Linker > Input > Additional Dependencies` بروید.
    • کتابخانه‌های SFML مورد نیاز را اضافه کنید. Debug: sfml-graphics.lib sfml-window.lib sfml-system.lib

مرحله ۴: کپی کردن فایل‌های DLL

فایل‌های DLL مورد نیاز SFML (که در پوشه `bin` کتابخانه SFML قرار دارند، مثلاً `C:\SFML-2.5.1\bin`) را به پوشه خروجی پروژه خود کپی کنید (معمولاً پوشه `Debug` یا `Release` در کنار فایل اجرایی برنامه).

مرحله ۵: تست نصب

یک کد ساده SFML (مانند مثال ایجاد پنجره) را در `main.cpp` بنویسید، پروژه را Build کرده و اجرا کنید. اگر پنجره SFML بدون خطا نمایش داده شد، نصب با موفقیت انجام شده است.

نکته مهم: مراحل دقیق ممکن است بسته به نسخه SFML، کامپایلر، و IDE شما کمی متفاوت باشد. همیشه به مستندات رسمی SFML برای راهنمایی دقیق‌تر مراجعه کنید.

ساختارهای پایه‌ای

این بخش به معرفی کلاس‌ها و توابع اساسی SFML می‌پردازد که برای ایجاد پنجره، مدیریت آن و نمایش محتوا استفاده می‌شوند.

تابع: sf::RenderWindow

کاربرد: ایجاد و مدیریت پنجره‌ی گرافیکی.

توضیح: این کلاس پنجره‌ای باز می‌کند که می‌توان در آن اشیاء گرافیکی رسم کرد. این پنجره نقطه شروع هر برنامه گرافیکی SFML است.

ساختار تابع:
sf::RenderWindow window(sf::VideoMode(width, height), "Title");
نمونه کد:
#include <SFML/Graphics.hpp>

int main()
{
    // یک پنجره با ابعاد 800x600 پیکسل و عنوان "SFML Window Example" ایجاد می‌کند.
    sf::RenderWindow window(sf::VideoMode(800, 600), "SFML Window Example");

    // حلقه اصلی برنامه که تا زمانی که پنجره باز است ادامه دارد.
    while (window.isOpen())
    {
        sf::Event event; // یک شیء رویداد برای دریافت ورودی‌ها
        // رویدادها را از صف رویدادها دریافت می‌کند.
        while (window.pollEvent(event))
        {
            // اگر رویداد از نوع بسته شدن پنجره باشد، پنجره را می‌بندد.
            if (event.type == sf::Event::Closed)
                window.close();
        }

        // پنجره را با رنگ مشکی پاک می‌کند.
        window.clear(sf::Color::Black);

        // در اینجا می‌توان اشیاء گرافیکی را رسم کرد.

        // محتوای پنجره را نمایش می‌دهد.
        window.display();
    }
    return 0;
}
خروجی اجرا شده:

یک پنجره با پس‌زمینه مشکی، با ابعاد ۸۰۰ در ۶۰۰ پیکسل و با عنوان "SFML Window Example" در بالای پنجره نمایش داده می‌شود. این پنجره تا زمانی که کاربر آن را ببندد (مثلاً با کلیک روی دکمه X) باز باقی می‌ماند.

تابع: window.draw()

کاربرد: رسم یک شیء قابل رسم (Drawable) روی پنجره.

توضیح: این تابع یک شیء گرافیکی (مانند شکل، متن، یا اسپریت) را به بافر پشتی (back buffer) پنجره اضافه می‌کند. این اشیاء تا زمانی که window.display() فراخوانی نشود، روی صفحه نمایش داده نمی‌شوند.

ساختار تابع:
void sf::RenderTarget::draw(const sf::Drawable& drawable, const sf::RenderStates& states = sf::RenderStates::Default);
// مثال استفاده:
sf::CircleShape circle(50.f);
window.draw(circle);
نمونه کد:
#include <SFML/Graphics.hpp>

int main()
{
    sf::RenderWindow window(sf::VideoMode(400, 300), "Draw Example");
    sf::CircleShape circle(50.f); // دایره با شعاع 50
    circle.setFillColor(sf::Color::Green); // رنگ سبز
    circle.setPosition(100.f, 100.f); // موقعیت دایره

    while (window.isOpen())
    {
        sf::Event event;
        while (window.pollEvent(event))
        {
            if (event.type == sf::Event::Closed)
                window.close();
        }
        window.clear(); // پاک کردن پنجره (با رنگ مشکی پیش‌فرض)
        window.draw(circle); // رسم دایره
        window.display();
    }
    return 0;
}
خروجی اجرا شده:

یک پنجره با پس‌زمینه مشکی و عنوان "Draw Example" نمایش داده می‌شود که در آن یک دایره سبز رنگ با شعاع ۵۰ پیکسل در موقعیت (۱۰۰, ۱۰۰) رسم شده است.

Draw Example

تابع: window.display()

کاربرد: نمایش محتوای رسم شده روی صفحه.

توضیح: این تابع محتوای بافر پشتی (که با window.draw() پر شده است) را به بافر جلویی (front buffer) منتقل می‌کند و باعث می‌شود تغییرات روی صفحه نمایش داده شوند. این تابع باید در انتهای هر فریم از حلقه اصلی برنامه فراخوانی شود.

void sf::Window::display();
نمونه کد:

(در مثال تابع window.draw() و سایر مثال‌های ایجاد پنجره، این تابع برای نمایش نهایی استفاده شده است.)

خروجی اجرا شده:

اشکال گرافیکی

SFML کلاس‌های مختلفی برای رسم اشکال هندسی ساده مانند دایره، مستطیل، و چندضلعی‌های دلخواه ارائه می‌دهد. این بخش به معرفی این کلاس‌ها و توابع مرتبط با آن‌ها می‌پردازد.

کلاس: sf::CircleShape

کاربرد: رسم دایره.

توضیح: این کلاس برای ایجاد و نمایش یک دایره (یا بیضی اگر مقیاس‌دهی شود) استفاده می‌شود. می‌توان شعاع، تعداد نقاط تشکیل‌دهنده (برای صافی لبه‌ها)، رنگ، موقعیت و سایر ویژگی‌های آن را تنظیم کرد.

ساختار تابع (سازنده):
sf::CircleShape(float radius = 0, std::size_t pointCount = 30);
نمونه کد:
#include <SFML/Graphics.hpp>

int main()
{
    sf::RenderWindow window(sf::VideoMode(400, 300), "CircleShape Example");
    sf::CircleShape circle(50.f); // دایره با شعاع 50
    circle.setFillColor(sf::Color::Red);
    circle.setPosition(150.f, 100.f);

    while (window.isOpen())
    {
        sf::Event event;
        while (window.pollEvent(event))
        {
            if (event.type == sf::Event::Closed)
                window.close();
        }
        window.clear();
        window.draw(circle);
        window.display();
    }
    return 0;
}
خروجی اجرا شده:

یک پنجره با عنوان "CircleShape Example" نمایش داده می‌شود که در آن یک دایره قرمز رنگ با شعاع ۵۰ پیکسل در موقعیت (۱۵۰, ۱۰۰) قرار دارد.

کلاس: sf::ConvexShape

کاربرد: رسم چندضلعی‌های دلخواه و مثلث.

توضیح: این کلاس برای ایجاد اشکال چندضلعی با تعداد نقاط دلخواه استفاده می‌شود. برای رسم مثلث، کافی است ۳ نقطه تعریف کنید.

ساختار تابع (سازنده):
sf::ConvexShape(std::size_t pointCount = 0);
نمونه کد (مثلث):
#include <SFML/Graphics.hpp>

int main()
{
    sf::RenderWindow window(sf::VideoMode(400, 300), "Triangle Example");
    
    // ایجاد یک مثلث
    sf::ConvexShape triangle;
    triangle.setPointCount(3); // تعیین 3 نقطه برای مثلث
    triangle.setPoint(0, sf::Vector2f(100.f, 0.f));   // نقطه بالایی
    triangle.setPoint(1, sf::Vector2f(0.f, 200.f));   // نقطه پایین چپ
    triangle.setPoint(2, sf::Vector2f(200.f, 200.f)); // نقطه پایین راست
    triangle.setFillColor(sf::Color::Yellow);
    triangle.setPosition(100.f, 50.f);

    while (window.isOpen())
    {
        sf::Event event;
        while (window.pollEvent(event))
        {
            if (event.type == sf::Event::Closed)
                window.close();
        }
        window.clear();
        window.draw(triangle);
        window.display();
    }
    return 0;
}
نمونه کد (پنج ضلعی):
#include <SFML/Graphics.hpp>
#include <cmath> // برای تابع sin و cos

int main()
{
    sf::RenderWindow window(sf::VideoMode(400, 300), "Pentagon Example");
    
    // ایجاد یک پنج ضلعی
    sf::ConvexShape pentagon;
    pentagon.setPointCount(5);
    
    // محاسبه نقاط پنج ضلعی با استفاده از فرمول‌های دایره
    float radius = 50.f;
    for (int i = 0; i < 5; ++i)
    {
        float angle = i * 2 * 3.14159f / 5; // زاویه بر حسب رادیان
        pentagon.setPoint(i, sf::Vector2f(
            radius * std::cos(angle),
            radius * std::sin(angle)
        ));
    }
    
    pentagon.setFillColor(sf::Color::Magenta);
    pentagon.setPosition(200.f, 150.f);

    while (window.isOpen())
    {
        sf::Event event;
        while (window.pollEvent(event))
        {
            if (event.type == sf::Event::Closed)
                window.close();
        }
        window.clear();
        window.draw(pentagon);
        window.display();
    }
    return 0;
}
